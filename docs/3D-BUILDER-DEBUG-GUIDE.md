## Enabling Debug Mode

### Console Commands
```javascript
// Enable debugging (in browser console)
window.enableDebug('debug');  // Options: 'error', 'warn', 'info', 'debug', 'trace'

// Disable debugging
window.disableDebug();

// Access debug service directly
window.proshotDebug.getLogs();
window.proshotDebug.exportLogs();
```

### "Generation Failed"
1. Check `generationAPICall` log â†’ Was prompt sent?

3. Check console for network errors
4. Verify API key in `.env`

### "Scene Not Rendering"
1. Check `renderSceneDetected` log â†’ Found scene/camera/renderer?
2. Check `runtimeError` log â†’ JS errors in generated code?
3. Look at iframe console for Three.js errors
4. Verify import map URLs are accessible

### "Boolean Operation Failed"
1. Check `panelBooleanStarted` log â†’ Target ID correct?
2. Check `messageToIframe` log â†’ Message sent?
3. Check `panelBooleanCompleted` log â†’ Success status?
4. Ensure both objects are valid meshes (not groups)

### "Parameters Not Working"
1. Check `guiConfigReceived` log â†’ Controls received?
2. Verify AI code uses `new window.GUI().add()`
3. Check `panelParameterChanged` log â†’ Values changing?
4. Check `messageToIframe('updateParam')` log

### "Export Not Working"
1. Check `exportStarted` log â†’ Format correct?
2. Check `exportCompleted` log â†’ Received?
3. Look at iframe console for export errors
4. Verify scene has exportable meshes
